Nouveau service BeaverKiosk

BeaverTask

BeaverTask centralise la gestion des tâches pour les équipes locales. Il propose une liste d'actions claire, un calendrier des échéances et un partage rapide grâce aux codes QR, le tout propulsé par une base de données SQLite hors ligne.

To-Do List

Visualisez l'ensemble des tâches dans un flux principal. Chaque carte affiche le titre, la priorité et l'échéance pour aider les équipes à prioriser les actions critiques en un clin d'œil.

Add Task Menu

Un formulaire tactile simplifié permet de définir un responsable, une priorité et une date limite. La date de création et l'identifiant BeaverTask sont générés automatiquement pour assurer la traçabilité.

Calendar Tab

Les tâches sont projetées sur un calendrier dédié, offrant une vue d'ensemble des échéances. Les filtres par statut facilitent la planification et la répartition de la charge de travail.

SQLite Local Storage

Toutes les informations sont persistées dans SQLite, garantissant des opérations rapides et un fonctionnement hors ligne complet sur les bornes BeaverKiosk.

Task Sharing

Chaque tâche génère un code QR. Un simple scan permet d'importer les détails sur un autre kiosk ou appareil, simplifiant la collaboration inter-sites.

Structure des données BeaverTask

Chaque entrée est identifiée par un code unique respectant le format BV-TS-001. Les champs ci-dessous couvrent l'ensemble des métadonnées nécessaires pour suivre, filtrer et partager les tâches.

Task Title

Description concise de l'action à réaliser.

Assignee

Personne ou équipe responsable de l'exécution.

Created Date

Date d'enregistrement automatique pour l'historique.

Due Date

Échéance cible, utilisée pour le calendrier et les alertes.

Priority

Indice d'urgence (High, Medium, Low) pour organiser le travail.

Status

État courant (Not Started, In Progress, Completed) pour le suivi.

Notes & Links

Espace libre pour instructions, commentaires ou URL utiles.

Image

Pièce jointe optionnelle (photo de terrain, croquis, etc.).

Unique ID

Code généré automatiquement au format BV-TS-[###] pour tracer chaque tâche.
